### Steps to run 
Clone this repository:

```sh
git clone https://github.com/mangya/reactdemo.git
cd reactdemo
```

Install dependencies:

```sh
npm install
```

Start the project at [`http://localhost:3000`](http://localhost:3000).

```sh
npm start
```
